# TP3 - Application Stars

## Description

L'application **Stars** permet aux utilisateurs de découvrir et de consulter une liste d'acteurs célèbres avec leurs photos, noms et notes. Elle propose une interface simple et intuitive, idéale pour une première exploration des célébrités. 

## Fonctionnalités

- **Écran de chargement (Splash Screen)** : Un écran de démarrage avec le logo de l'application avant de charger la liste des acteurs.
- **Liste des Acteurs** : Affiche une liste d'acteurs avec leur photo, nom et évaluation (sous forme d'étoiles).
- **Recherche d'Acteurs** : Permet de filtrer la liste des acteurs via une barre de recherche en temps réel.
- **Partage d'Informations** : Option de partager les informations d'un acteur via d'autres applications.

## Technologies Utilisées

- **Langage** : Java
- **Interface Utilisateur** : XML
- **Framework** : Android SDK
- **Bibliothèque d'Images** : Glide
- **IDE** : Android Studio
